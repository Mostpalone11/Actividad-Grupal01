Descripción del proyecto:

Esta actividad contiene una serie de ejercicios en Python, diseñados para practicar:

1- Entrada y salida de datos (input, print)

2- Uso de variables y operadores

3- Estructuras de control (if, elif, else)

4- Bucles (for, while)

5- Funciones básicas y modularidad

6- Generación de números aleatorios (random)

Cada ejercicio incluye comentarios explicativos sobre el propósito del programa, 
el uso de variables y la lógica implementada.
Al finalizar, el programa debe estar correctamente documentado y
 listo para ser utilizado en el aula.
